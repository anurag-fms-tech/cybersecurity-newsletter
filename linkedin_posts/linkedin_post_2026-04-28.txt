1. Checkmarx Confirms GitHub Repository Data Posted on Dark Web After March 23 Attack
Threat: Data Breach | Severity: 8/10
Checkmarx disclosed a cybercrime group published its data on the dark web, originating from a GitHub repository compromised in a March 2026 supply chain attack. The data includes source code, employee database, API keys, and credentials. The company is investigating and has locked down the affected repository, stating it will notify customers if their information was involved.

4. PhantomCore Exploits TrueConf Vulnerabilities to Breach Russian Networks
Threat: Vulnerability | Severity: 8/10
A pro-Ukrainian hacktivist group, PhantomCore, has been linked to attacks on Russian organizations using vulnerable TrueConf video conferencing software, exploiting three unpatched vulnerabilities since September 2025. The group has used these exploits to gain network access, steal data, and deploy ransomware, with some attacks remaining undetected for extended periods due to the group's use of in-house offensive tools and strong stealth capabilities.

5. Researchers Uncover 73 Fake VS Code Extensions Delivering GlassWorm v2 Malware
Threat: Malware | Severity: 8/10
Cybersecurity researchers have flagged 73 cloned Microsoft Visual Studio Code extensions on the Open VSX repository linked to a malware campaign dubbed GlassWorm. Six extensions have been confirmed malicious, while 67 are seemingly harmless sleeper packages aimed at building user trust, before manifesting their true intent through updates. Over 320 artifacts have been linked to the campaign since December 21, 2025.
