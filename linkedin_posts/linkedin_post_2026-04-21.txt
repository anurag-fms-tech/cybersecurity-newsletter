1. SGLang CVE-2026-5760 (CVSS 9.8) Enables RCE via Malicious GGUF Model Files
Threat: Vulnerability | Severity: 8

Explanation: 
A CVSS (Common Vulnerability Scoring System) score of 9.8 is extremely high, indicating a highly severe vulnerability. However, the fact that it was publicly disclosed in 2026, rather than being a more recent discovery, suggests a lower risk of active exploitation./10
A critical vulnerability (CVE-2026-5760) in SGLang, a high-performance serving framework, has been disclosed, allowing remote code execution with a CVSS score of 9.8. The vulnerability exploits a command injection through a specially crafted GPT-Generated Unified Format model file, and patching involves using ImmutableSandboxedEnvironment for rendering chat templates. A similar vulnerability was fixed in vLLM and llama_cpp_python in 2025 and 2024, respectively.

4. Anthropic MCP Design Vulnerability Enables RCE, Threatening AI Supply Chain
Threat: Vulnerability | Severity: 9/10
Cybersecurity researchers have discovered a critical weakness in the Model Context Protocol (MCP) that enables attackers to remotely execute code and access sensitive data. The flaw, found in over 7,000 servers and 150 million downloads, affects popular AI projects and exposes users to command execution risks. Anthropic declined to modify the protocol, forcing developers to patch individual implementations to mitigate the threat.

5. Researchers Detect ZionSiphon Malware Targeting Israeli Water, Desalination OT Systems
Threat: Malware | Severity: 8/10
Cybersecurity researchers have identified a new malware, ZionSiphon, specifically designed to target Israeli water treatment and desalination systems. It combines privilege escalation, persistence, and sabotage capabilities, embedding messages supporting Iran, Palestine, and Yemen. Though unfinished and currently undelivered, ZionSiphon demonstrates an experimenting threat actor's efforts to manipulate operational technologies and propagate via removable media, highlighting evolving critical infrastructure threats.
