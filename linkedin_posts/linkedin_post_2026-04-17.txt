Cybersecurity Intelligence Brief

1. Apache ActiveMQ CVE-2026-34197 Added to CISA KEV Amid Active Exploitation
Threat: Vulnerability | Severity: 10/10
A high-severity security flaw (CVE-2026-34197) in Apache ActiveMQ Classic has been exploited in the wild, prompting CISA to add it to its Known Exploited Vulnerabilities catalog. The vulnerability, discovered after 13 years, allows attackers to execute arbitrary code and requires urgent fixes by April 30, 2026, to prevent potential data exfiltration or disruption.

2. Newly Discovered PowMix Botnet Hits Czech Workers Using Randomized C2 Traffic
Threat: Vulnerability | Severity: 8/10
Cybersecurity researchers have uncovered a malicious botnet called PowMix targeting the Czech workforce since at least December 2025. The botnet uses randomized command-and-control beaconing intervals and mimics legitimate URLs to evade detection. It drops a multi-stage infection chain and can remotely update its configuration, facilitating remote access, reconnaissance, and code execution with decoy documents to distract victims.

5. Cisco Patches Four Critical Identity Services, Webex Flaws Enabling Code Execution
Threat: Vulnerability | Severity: 9/10
Cisco has released patches to address four critical vulnerabilities in Identity Services and Webex Services. The flaws, including CVE-2026-20147, CVE-2026-20180, and CVE-2026-20186, could lead to arbitrary code execution, user impersonation, and denial-of-service conditions. Customers are advised to update to the latest version for optimal protection.
