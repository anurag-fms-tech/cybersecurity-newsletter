Researchers Uncover Pre-Stuxnet ‘fast16’ Malware Targeting Engineering Software

Threat: Malware
Severity: 8/10

Researchers have discovered a new Lua-based malware called Fast16, created in 2005, which aimed to sabotage Iran's nuclear program by altering high-precision calculations. It predates the notorious Stuxnet worm by at least five years, making it the first documented Windows malware to embed a Lua engine. The malware's core logic resides in the Lua bytecode, with the binary referencing a kernel driver responsible for intercepting and modifying executable code.
