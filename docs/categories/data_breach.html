Student Loan Breach Exposes 2.5M Records

Severity: 8/10

A data breach exposed personal data of 2.5 million student loan account holders. Nelnet Servicing, a servicing system provider for EdFinancial and the Oklahoma Student Loan Authority, revealed the breach on July 21, 2022, after a cybersecurity team discovered suspicious activity. The exposed information includes names, addresses, email addresses, phone numbers, and social security numbers, but no financial information was compromised.

FBI Extracts Deleted Signal Messages from iPhone Notification Database

Severity: 8/10

The FBI was able to extract Signal message data from an iPhone after the app was deleted, using the device's push notification database. However, Apple has since patched this vulnerability, which was discovered during a trial and highlights the importance of turning on Signal's setting to block message content in push notifications.
