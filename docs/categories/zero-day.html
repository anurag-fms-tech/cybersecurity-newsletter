Nine-Year-Old Zero-Day Flaw in Linux Kernel Discovered by AI-Equipped Security Researcher

Severity: 8/10

A nine-year-old, high-security zero-day vulnerability, 'Copy Fail', has been discovered in the Linux kernel since 2017. With the help of AI-powered tool Xint Code, vulnerability researcher Taeyang Lee found the flaw, which lets attackers gain root access to the kernel, putting shared systems and container clusters at risk. A patch has been released, and the vulnerability is addressed as CVE-2026-31431.
