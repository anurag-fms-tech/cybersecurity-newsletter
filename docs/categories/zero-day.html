Nine-Year-Old Zero-Day Flaw in Linux Kernel Discovered by AI-Equipped Security Researcher

Severity: 9/10

A 9-year-old high-security zero-day vulnerability, named 'Copy Fail', has been discovered in the Linux kernel using AI-driven analysis tool Xint.io. The flaw, discovered by Taeyang Lee, allows an unprivileged local user to gain root access to a machine with physical access. A patch has been released and the vulnerability affects all Linux distributions shipped since 2017, posing a risk to shared systems, container clusters, and similar environments.
